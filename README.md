# F1-Style-Shift-Light-Microcontroller
I built a shift light system for my car, similar to what you see on F1 cars or digital gauges. It plugs into the OBD-II port under the dashboard and reads live engine RPM directly from the car’s CAN bus network which is the internal network all the car’s computers communicate over. That data is processed by the Arduino Mega which drives an LED strip that progressively lights up as you rev the engine and strobes when you hit your target shift RPM, along with an LCD screen displaying the exact RPM in real time
